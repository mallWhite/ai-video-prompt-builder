### 具体实现建议

#### 1. 前端优化（无需API）
在调用API之前，先进行本地智能分析：

```javascript
// 智能提取文案要素
function analyzeText(text) {
  return {
    subject: extractSubject(text),      // 主体：工人/机器/团队
    action: extractAction(text),        // 动作：操作/检查/协作
    emotion: extractEmotion(text),      // 情绪：专注/紧张/自豪
    scene: extractScene(text),          // 场景：井下/港口/工厂
    keywords: extractKeywords(text)     // 关键词数组
  };
}
```

#### 2. API调用策略

**方式A：纯前端调用（CORS限制较多）**
- 智谱、讯飞支持CORS
- 需要暴露API Key（安全性低）
- 适合个人使用

**方式B：添加简单后端（推荐）**
- 使用 Vercel/Cloudflare Workers 免费托管
- 隐藏API Key
- 可添加缓存、限流

**方式C：用户自备API Key**
- 让用户输入自己的API Key
- 纯前端调用
- 最安全、零成本

#### 3. 推荐实现路径

**阶段1：用户自备Key模式（立即实现）**
```
- 在界面添加"API Key设置"区域
- 支持智谱、讯飞、Gemini等多种API
- 用户输入Key后，前端直接调用
- 零成本、可立即上线
```

**阶段2：添加简单后端（后续优化）**
```
- 使用 Cloudflare Workers（每天10万次免费请求）
- 或 Vercel Edge Functions（免费额度充足）
```

## 🔧 技术实现代码示例

### 智谱 GLM-4-Flash 调用示例

```javascript
async function generatePromptWithAI(text, sceneType) {
  const API_KEY = localStorage.getItem('zhipu_api_key');
  if (!API_KEY) {
    alert('请先设置智谱API Key');
    return;
  }

  const systemPrompt = `你是一位专业的AI视频提示词工程师，擅长将简短文案转化为详细的视频生成提示词。

要求：
1. 分析文案中的主体、动作、情绪、场景
2. 生成符合${sceneType}场景的专业提示词
3. 包含：主体描述、动作细节、环境氛围、镜头建议、光线效果、画质要求
4. 写实风格，符合物理规律，避免夸张
5. 输出中文提示词，适合可灵/豆包等国产AI视频工具`;

  const response = await fetch('https://open.bigmodel.cn/api/paas/v4/chat/completions', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      model: 'glm-4-flash',
      messages: [
        { role: 'system', content: systemPrompt },
        { role: 'user', content: `文案：${text}` }
      ],
      temperature: 0.7,
      max_tokens: 800
    })
  });

  const data = await response.json();
  return data.choices[0].message.content;
}
```

### 多平台适配

```javascript
function adaptForPlatform(prompt, platform) {
  switch(platform) {
    case 'kling':
      return prompt; // 可灵直接用中文
    case 'doubao':
      return prompt.replace(/【.*?】/g, '').replace(/\n/g, '，'); // 豆包偏好简洁段落
    case 'sora':
    case 'runway':
      return translateToEnglish(prompt); // 需要翻译为英文
    default:
      return prompt;
  }
}
```
