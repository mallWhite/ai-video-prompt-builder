# 2026-04-17 工作记录

## AI智能提示词生成功能上线

### 新增功能

#### 1. AI模式切换
- 在文案输入面板上方新增"AI智能增强"设置面板
- 支持切换"本地模板"和"AI智能"两种生成模式
- 切换时自动显示/隐藏API Key配置区域

#### 2. 多平台API支持
集成三个免费AI API：

| 平台 | 模型 | 特点 |
|------|------|------|
| **智谱** | GLM-4-Flash | 永久免费，中文最强，首选推荐 |
| **讯飞** | spark-lite | 无限Token，备选方案 |
| **Google** | Gemini 1.5 Flash | RPM=15，英文优化 |

#### 3. 智能提示词生成
- AI根据用户文案智能分析主体、动作、环境、镜头、光线等要素
- 生成更专业、更详细的视频提示词
- 自动融合用户选择的参数（镜头类型、光线、风格等）

#### 4. API Key管理
- 本地存储API Key（localStorage），不上传到服务器
- 支持为不同平台分别配置Key
- Key显示为脱敏格式（前4位+****+后4位）
- 提供获取免费API Key的引导链接

#### 5. 用户体验优化
- 添加加载动画，AI生成时显示"AI正在分析文案..."
- 历史记录标注AI生成内容（🤖标记）
- 错误处理：未配置Key时自动展开配置面板

### 技术实现

#### API调用架构
```
用户文案 + 场景参数
    ↓
构建专业Prompt模板
    ↓
调用智谱/讯飞/Gemini API
    ↓
AI生成专业提示词
    ↓
适配不同平台格式
    ↓
输出最终结果
```

#### Prompt模板设计
```
你是一位专业的AI视频提示词工程师...

【用户文案】{text}
【场景类型】{scene}
【参考参数】镜头、光线、风格、情绪等

【任务要求】
1. 主体描述
2. 动作细节
3. 环境氛围
4. 镜头语言
5. 技术参数
6. 写实要求

【输出格式】
直接输出可用于AI视频工具的中文提示词
```

### 使用流程

1. **开启AI模式** — 点击切换开关启用AI智能生成
2. **配置API Key** — 选择平台并输入API Key（首次使用）
3. **输入文案** — 在文案框中输入原始描述
4. **调整参数**（可选）— 选择镜头、光线、风格等作为参考
5. **生成提示词** — AI分析文案并生成专业提示词
6. **复制使用** — 复制到可灵/豆包/Runway/Sora等工具
