# AI 视频提示词生成框架 · 多场景版

一个基于浏览器的 AI 视频提示词生成工具，专为工业场景设计，支持煤矿、港口、工厂、建筑工地、铁路等多种场景。

## 🎬 在线访问

**GitHub Pages**: https://[你的用户名].github.io/ai-video-prompt-builder/

## ✨ 功能特点

- **5 种工业场景**：煤矿 ⛏️、港口 🚢、工厂 🏭、建筑工地 🏗️、铁路 🚄
- **多平台支持**：Sora、Runway Gen-3、可灵 Kling、通义万象、豆包
- **现实逻辑约束**：所有提示词均强调物理准确、写实风格、符合行业规范
- **场景专属标签**：每个场景 12 个专属元素标签
- **历史记录**：本地存储，最多保存 20 条记录

## 🚀 本地使用

直接用浏览器打开 `coal-mine-prompt-builder.html` 即可使用，无需服务器。

## 📁 文件说明

- `coal-mine-prompt-builder.html` - 主程序文件（单文件，包含所有 HTML/CSS/JS）

## 📝 使用说明

1. 选择场景（顶部标签切换）
2. 配置视频参数（镜头、光线、风格、情绪等）
3. 输入你的文案
4. 点击「生成视频提示词」
5. 复制提示词到 AI 视频工具中使用

## 🔧 技术说明

- 纯前端实现，无需后端
- 使用 localStorage 存储历史记录
- 响应式设计，支持移动端
